# Dashboard Ganadería Regenerativa

Este repositorio contiene el dashboard de métricas de redes sociales de GanaderíaRegenerativa.com.

## Estructura
- `index.html` — Dashboard principal (no modificar a menos que se pida)
- `data/dashboard-data.json` — Base de datos de métricas (se actualiza semanalmente)
- `README.md` — Documentación

## Tarea principal
Cuando recibas un archivo `dashboard-data.json` actualizado:
1. Reemplaza `data/dashboard-data.json` con el nuevo archivo
2. Haz commit con mensaje: "Actualización semana YYYY-WXX"
3. Haz push a origin main

## Reglas
- Nunca modifiques index.html sin instrucción explícita
- El JSON debe ser válido antes de hacer commit
- Siempre verifica que el JSON tenga la estructura correcta antes de subir
